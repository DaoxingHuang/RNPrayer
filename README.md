# RNPrayer

> 项目介绍

## Installing / Getting started

环境搭建：

> 参考 [RN环境搭建文档](https://github.com/DaoxingHuang/RNPrayer/blob/main/RN%E7%8E%AF%E5%A2%83%E6%90%AD%E5%BB%BA.md)

快速开始：

```shell
// 安装依赖，建议使用yarn
yarn install
// android
yarn android
// iOS
yarn ios
// bundlejs
yarn start
```

## Developing

### Built With

* node: >=14.0.0
* React: 18.0.0
* React-native: 0.69.4
* react-navigation: 6.x
* typescript: 4.4.4

### Prerequisites
- node
- watchman
- Xcode
- CocoaPods
- JDK -> 11
- Android SDK套件

### Setting up Dev

```shell
git clone https://github.com/DaoxingHuang/RNPrayer.git
cd RNPrayer/
yarn install
```

### Building Bundle (dev)

```shell
yarn start
```
